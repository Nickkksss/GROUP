Panda_Userbot adalah userbot Telegram modular yang berjalan di Python3 dengan database sqlalchemy.

Berbasis [Cat](https://github.com/sandy1709/catuserbot) ‚û° [Ultroid](https://github.com/TeamUltroid/Ultroid) ‚û° [Userge](https://github.com/UsergeTeam/Userge) & [Uniborg](https://github.com/udf/uniborg) userbot. Saya membuat repository ini untuk memilih dan menambahkan beberapa modul yang saya butuhkan dengan banyak perubahan, fitur dan modul.

## Disclaimer
```
Saya tidak bertanggung jawab atas penyalahgunaan bot ini.
Bot ini dimaksudkan untuk bersenang-senang sekaligus membantu Anda
mengelola grup secara efisien dan mengotomatiskan beberapa hal yang membosankan.
Gunakan bot ini dengan risiko Anda
```
